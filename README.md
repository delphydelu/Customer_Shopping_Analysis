# 🛒 Customer Shopping Analysis

## 🔹 Project Overview
This project focuses on analyzing customer shopping behavior to uncover patterns in purchasing, spending habits, and customer preferences. The goal is to generate insights that can help businesses improve sales and customer experience.

---

## 🔹 Tools & Technologies
- Python (Pandas, NumPy)
- Data Analysis
- Power Bi
- SQL

---

## 🔹 Objectives
- Understand customer purchasing patterns  
- Identify high-value customers  
- Analyze product/category performance  
- Discover trends in customer behavior  

---

## 🔹 Steps Performed
- Data Cleaning (handled missing values, removed duplicates)
- Data Transformation and formatting
- Exploratory Data Analysis (EDA)
- Aggregations and grouping for insights

---

## 🔹 Key Insights
- Certain product categories contribute more to overall revenue  
- High-spending customers form a small but valuable segment  
- Purchase frequency varies significantly across customers  
- Seasonal trends impact customer buying behavior  

---

## 🔹 Results
- Identified key revenue-driving customer segments  
- Improved understanding of customer purchase patterns  
- Generated insights to support better business decisions  

---

## 🔹 Project Structure
- data/ → dataset used for analysis  
- notebook/ → Python analysis file
- SQL/ → Data manipulation 
- Power Bi- / → charts and visualizations  

---

## 📌 Conclusion
This project demonstrates the use of data analysis techniques to extract meaningful insights from customer shopping data and support data-driven decision making.
